Brumby Code Checker

This is a simple redundancy code scanner that can be used to find and flag duplicated code. The code checker uses the SabbleCC compiler and a multi-dimensional similarity heuristic.

It functions by running three independent similarity algorithms and computes a weighted average of those three scores: cosine similarity, an ordered Naive 1-to-1 similarity, and a longest common subsequence algorithm. It tokenizes the code by renaming variables to an identifier, while discarding comments, white spaces and unknown tokens.

The application currently works on Java and C++.
